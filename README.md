# Profit_Analysis_Dashboard
Tableau dashboard analyzing profit ratios, sales trends, and product category performance across global regions (2016–2019).


## Overview
An interactive Tableau dashboard exploring profit and sales performance  across global regions (APAC, EMEA, LATAM, USCA) from 2016–2019.

## Dashboard Preview
[![Profit Analysis Dashboard](https://public.tableau.com/views/Tableau_Part7_17721637470450/ProfitAnalysisDashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link) 
> Click the image to view the live, interactive dashboard on Tableau Public.

## Key Insights
- Overall profit ratio of 11.6% in 2019, with sales up 26.3% YoY.
- **Tables** is the only product sub-category operating at a loss
- September 2017 saw peak sales (~$140K), with profit peaking around the same period
- Technology and Office Supplies consistently outperform Furniture in profit ratio

## Features
- Region and year filter buttons for dynamic data exploration
- Profit ratio vs. average benchmark comparison by sub-category
- Dual-axis Profit to Sales trend line with monthly granularity

## Tools Used
- Tableau Desktop / Tableau Public
- Dataset: Superstore Sales
